# Simplify9

**We design and build scalable, intelligent digital solutions that help businesses automate, grow, and innovate faster.**

Founded in 2014, Simplify9 is an AI driven software design and consultancy company delivering software engineering, cloud, and intelligent automation. We've shipped 200+ projects across logistics, fintech, insurance, tourism, and retail — processing 100M+ transactions for 50+ clients in 15+ countries.

## What we do

- **AI & Intelligent Automation** — NLP, computer vision & OCR, predictive analytics, MLOps, RPA
- **Software Engineering** — web & mobile apps, backend systems & APIs, integrations, legacy modernization
- **Cloud & DevSecOps** — cloud migration, IaC, CI/CD, Kubernetes, security & compliance (Azure · AWS · GCP · OCI)
- **Consulting & Tech Advisory** — architecture reviews, digital roadmapping, AI & data strategy
- **Managed Services** — monitoring & support, infrastructure management, dedicated team augmentation

## Featured open source

We maintain a family of reusable .NET libraries and tools that power our products.

| Project | Description |
| --- | --- |
| [Bitween](https://github.com/simplify9/Bitween-api) | Protocol-agnostic integration middleware for connecting systems and services |
| [SW-Serverless](https://github.com/simplify9/SW-Serverless) | Serverless execution framework for .NET |
| [SW-PrimitiveTypes](https://github.com/simplify9/SW-PrimitiveTypes) | Reusable base types and interfaces for common patterns |
| [SW-Mtm-api](https://github.com/simplify9/SW-Mtm-api) | Multi-tenant user and tenant management microservice |
| [SW-CloudFiles](https://github.com/simplify9/SW-CloudFiles) | S3-backed file operations with streaming and ASP.NET Core DI |
| [SW-CqApi](https://github.com/simplify9/SW-CqApi) | CQRS-style API framework |

Browse all of our [`SW-*` libraries](https://github.com/orgs/simplify9/repositories?q=SW-) — logging, EF Core extensions, HTTP helpers, Excel import/export, Blazor form components, and more.
